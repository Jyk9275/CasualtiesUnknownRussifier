# Русификатор для "Casualties: Unknown".

Примечание: мод не доделан, могут быть баги.

## Как установить:

1. Установите BepInEx(https://github.com/BepInEx/BepInEx).
2. Скачайте мод.
3. Распакуйте мод в папку игры(где находится .exe файл).

## Не локализировано:
* Анимация последнего рывка(не могу сделать);
* Реплики персонажей с упоминанием конечности;
* Качества предметов в инвентаре и в меню крафта.
